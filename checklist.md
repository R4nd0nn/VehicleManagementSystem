# 集装箱卡车运输企业内部管理系统 - 功能清单

√：已完成 W:正在进行 
## 模块1：订单管理

- [W] 订单录入：客户信息、箱型/箱量/箱号、提箱点/还箱点/送货地址、船名航次、截关时间、运输要求、加急标识，自动生成唯一运单号
- [W] 订单状态：待派车、已派车、运输中、已完成、异常、已取消、已结算（7大状态）
- [W] 订单操作：新增、修改、复制、作废、加急标记、批量查询、数据导出

## 模块2：调度管理

- [ ] 基础信息管理：车辆档案（车牌号、车型、状态）、司机档案（姓名、电话、驾驶证、绑定车辆）
- [ ] 派单功能：手动派单、智能推荐派单、批量派单、紧急插单、改派、撤单
- [ ] 调度工作台：今日任务、待派订单、在途车辆、异常预警展示
- [ ] 任务推送：派单后实时推送至司机移动端

## 模块3：司机移动端

- [ ] 任务管理：待执行、进行中、已完成任务展示
- [ ] 现场签到：提箱、到厂、还箱三大签到节点
- [ ] 资料上传：箱况、封条、闸口小票、回单拍照
- [ ] 定位跟踪：GPS位置自动上报
- [ ] 异常上报：无箱、故障、延误、箱损货损 + 佐证材料
- [ ] 费用收取：等候费、压车费、放空费申请

## 模块4：集装箱管理

- [ ] 箱况跟踪：箱号档案，空箱/重箱状态，提箱/还箱时间
- [ ] 预警提醒：超期箱、滞箱自动预警
- [ ] 异常登记：箱损、污箱、修箱，关联运单
- [ ] 扩展对接：预留码头/堆场EDI接口

## 模块5：在途监控模块

- [ ] 实时定位与轨迹查看（支持历史回放）
- [ ] 异常停留/超时未到自动预警
- [ ] 预估到达时间推送

## 模块6：费用管理

- [ ] 客户应收：基础运费 + 提卸费、等候费、过夜费、放空费、压车费 + 自定义杂费
- [ ] 车队应付：司机提成、路桥费、停车费等运营成本
- [ ] 费用管控：规则预设、自动计算、人工审核/调整，费用与订单绑定

## 模块7：财务管理

- [ ] 应收/应付账单生成与汇总
- [ ] 对账单导出、打印
- [ ] 开票信息与收付款记录登记
- [ ] 收支分类统计（单票、单车、单客户利润）

## 模块8：异常管理

- [ ] 异常类型：无箱、还箱被退、故障、延误、箱损货损、客户投诉
- [ ] 异常全流程：上报→处理→备注→结案
- [ ] 异常统计报表

## 模块9：客户与车司管理（现阶段不用）

- [ ] 客户管理：档案、报价、历史订单、对账记录
- [ ] 车司管理：车辆保险/年检/维修，司机考勤/安全记录/工作量统计

## 模块10：数据报表与驾驶舱

- [ ] 核心数据：订单量、总营收、成本、净利润
- [ ] 绩效数据：车辆效率排行、司机工作量排行、订单完成率
- [ ] 异常数据：异常率、延误率、附加费频次
- [ ] 可视化图表：折线图、柱状图、饼图 + 明细导出

## 模块11：系统管理

- [ ] 用户账号与角色权限管理（数据隔离）
- [ ] 操作日志记录
- [ ] 数据备份与恢复
- [ ] 自定义配置（费用规则、编号规则、基础参数）
